# Weather-web-Application-
Developed a Weather Application using React.js, integration APIs to fetch real-time weather data. 

🌦️ Weather Web Application (React.js)

A responsive Weather Web Application built using React.js that fetches real-time weather data from external APIs. This app allows users to search for any city and view current weather conditions with a simple, clean, and interactive user interface.

🚀 Features

Search weather by city name

Real-time weather data using API integration

Displays temperature, humidity, wind speed, and weather condition

Responsive design for mobile and desktop

Clean and user-friendly UI

🛠️ Tech Stack

React.js

JavaScript (ES6+)

CSS

Weather API (e.g., OpenWeatherMap)

📌 How It Works

The application sends API requests based on the city entered by the user and dynamically updates the UI with the latest weather information received from the API.
